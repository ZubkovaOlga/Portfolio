 Разработала в Postman коллекцию запросов для API DummyAPI, охватив все основные операции для работы
с тремя ключевыми сущностями: пользователями, постами и комментариями. Создала последовательность запросов 
для управления пользователями — начиная с получения полного списка, затем создание нового пользователя с 
заполнением необходимых полей, получение конкретных данных пользователя по его ID, его обновление (например, изменение имени)
и завершив удалением пользователя, что важно для тестирования сценариев удаления и восстановления данных.

В целом,  создали полную коллекцию, которая позволяет протестировать весь CRUD (создание, чтение, обновление, удаление)
цикл для трех ключевых сущностей системы через API. Это значительно повышает эффективность автоматизированного тестирования,
помогает выявить возможные ошибки и убедиться в корректности работы API при различных сценариях взаимодействия.
